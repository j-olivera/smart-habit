# AGENTS.md - Smart Habit 

Este archivo actúa como el índice principal para el comportamiento, identidad y restricciones de los agentes de IA que operan en este proyecto. 

## Índice de Inteligencia Artificial

- [Identidad y Orquestación](docs/ia/memory/orchestrator.md) - Define quién es el orquestador principal, los sub-agentes disponibles y sus responsabilidades.
- [Arquitectura y Stack](docs/ia/memory/architecture.md) - Define las tecnologías, directorios base y la arquitectura del sistema.
- [Flujo de Autenticación](docs/ia/memory/auth-flow.md) - Detalla la seguridad, AuthGuard y Refresh Tokens.
- [Convenciones de Código](docs/ia/memory/conventions.md) - Reglas estrictas sobre Frontend (Angular), Backend (Java/Spring Boot), Docker, y Testing (Vitest/Playwright).
- [Protocolo de Operación](docs/ia/memory/protocol.md) - Límites absolutos, restricciones de seguridad y el protocolo de escalación.
- [Sub-agentes](docs/ia/agents/) - Definición de cada agente especializado (backend, frontend, db, qa).

> [!IMPORTANT]
> Todo agente que opere en este repositorio DEBE leer y respetar las directivas especificadas en estos documentos antes de ejecutar cualquier cambio.
